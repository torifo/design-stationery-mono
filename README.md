# BLANK STORE — design-stationery-mono

> **"Maximum focus through minimum tools."**

A design study of a simple, student-targeted stationery shop, executed under the **monochrome + one accent** theme. Pure white and ink black with a single vivid yellow as the only color signal.

## Overview

| | |
|---|---|
| **Brand** | BLANK STORE |
| **Theme** | Monochrome + 1 |
| **Live Site (planned)** | `design.stationery-mono.riumu.net` |

## Design Concept

- **Color**: White `#FFFFFF` × ink `#0A0A0A` × vivid yellow `#EAFF00`
- **Typography**: DM Sans (single face)
- **Aesthetic**: Lines, whitespace, zero decoration
- **UX**: 6 selected items, 4 category tiles, single quote, brief store info

## Tech

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN
- No framework, no build step
